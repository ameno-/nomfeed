Up and running in 60 seconds

Clone, install, and start feeding your agents. Works with everything you already use.

1

Clone & Install

$ git clone github.com/ameno-/nomfeed && bun install && bun link
2

Save anything

$ nomfeed add https://any-url.com
3

Search your library

$ nomfeed search "transformers"
+

Optional: LLM Extraction

$ export OPENROUTER_API_KEY=your-key
