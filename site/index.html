Everything you need to feed your agents

One CLI that handles URLs, videos, files, and notes — with optional LLM extraction.

🔗

URLs → Markdown

3-strategy cascade: Cloudflare text/markdown → Jina Reader (handles JS SPAs) → Readability fallback.

$ nomfeed add <url>
🎬

YouTube Transcripts

Full timestamped transcript via yt-dlp. Optional LLM extraction for structured insights.

$ nomfeed add <yt-url> --extract
📄

File Conversion

PDF, DOCX, XLSX, images, code files → markdown via Microsoft's markitdown.

$ nomfeed add ./report.pdf
📝

Quick Notes

Save text notes as markdown files instantly. Tag and search them later.

$ nomfeed note "idea..."
🔍

Full-Text Search

Search across all saved content — titles, bodies, extractions. Fast local search.

$ nomfeed search "query"
⚡

LLM Extraction

Fabric-inspired patterns extract wisdom, claims, references — not just a summary.

$ nomfeed extract <id>
🤖

MCP Server

Built-in MCP for Claude Desktop, Cursor, and other AI agents to search your library.

$ nomfeed mcp
🌐

Chrome Extension

One-click save from your browser. Right-click any page or use the popup. Optional extraction.

$ nomfeed serve
📡

HTTP API

RESTful API on localhost for integrations, automations, and external tools.

$ nomfeed serve --port 24242

Not a summary. Structured extraction.

Fabric-inspired LLM patterns that pull out ideas, insights, quotes, claims, and references. Each pattern = one focused API call. You pick which ones to run.

🧠 Extract Wisdom

Ideas, insights, quotes, habits, facts, references, and recommendations in structured sections.

Default

📑 Video Chapters

Timestamped chapter outline from any transcript. HH:MM:SS Topic Name format.

Default

⚖️ Analyze Claims

Truth claims with support/refutation evidence and A–F ratings. Logical fallacy detection.

Optional

📚 Extract References

Books, papers, tools, people, and projects mentioned. Categorized with context.

Optional

📋 Summarize

One-sentence summary + 10 main points + 5 takeaways. Clean and concise.

Optional

⭐ Rate Content

Quality scores: surprise, novelty, insight, practical value, wisdom. 0-10 per dimension.

Optional
+ Add your own patterns in ~/.nomfeed/patterns/<name>/system.md

Up and running in 60 seconds

No database. No Docker. No accounts. Just clone and go.

1. Clone & Install

$ git clone https://github.com/ameno-/nomfeed && cd nomfeed && bun install && bun link

2. Save something

$ nomfeed add https://any-url.com

3. Search your library

$ nomfeed search "query"

Optional: LLM Extraction

Bring your own OpenRouter key. Access any model.

$ export OPENROUTER_API_KEY=your-key
